DryadLinqQueryableSumAsQueryTSource Method (IQueryableTSource, ExpressionFuncTSource, NullableDouble)DryadLINQ documentation
Same as SumTSource(IQueryableTSource, ExpressionFuncTSource, NullableDouble), but returns an IQueryable<Double> containing a single element. Computes the sum of a set of nullable double values that is obtained by applying a function to each element of the input dataset.
